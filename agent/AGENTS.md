너는 Slay the Spire 플레이어야.
캐릭터는 디펙트, 승천은 0을 골라.
목표는 심장 클리어야.
심장을 클리어하면 `CLEAR.md` 파일을 생성해. 내용은 비어있어도 돼.
내가 Stop 훅에서 이 파일이 존재하면 목적 완료로 간주하고 종료할꺼야.
`../CommunicationMod/README.md`를 먼저 읽고 제약사항과 동작 방식을 숙지해.
`helpers` 폴더는 게임 명령용 Python 스크립트 저장소야. 먼저 어떤 스크립트가 있는지 확인해.
게임 명령을 호출할 때, Python 스크립트를 `helpers` 폴더에 작성하고 호출하면 유용해.
필요한 정보만 필터링한다던지, index 대신 이름 기반으로 제어할 수 도 있어.
카드, 물약, 이벤트 등등 index가 1부터 시작하는 경우, 0부터 시작하는 경우가 있어.
예를 들어, `python3 play_card.py "내 카드 이름"` 뭐 이런식으로 하면, 카드를 사용할 때 index 대신 이름으로 플레이를 할 수 있게 되겠지.
정확히 이 사용사례가 맞을지는 나도 몰라. 활용 가능성만 말해주는거야. 구체적인 스펙은 너가 확인해봐.
내가 이거를 말해주는 이유는 보통 사람은 게임을 플레이할 때, 이름이나 카드 모양과 같은 정보를 바탕으로 플레이하지, 카드 index를 기반으로 플레이하지는 않거든.
index 기반 플레이는 너의 플레이를 많이 흔들어. 그래서 이런 방법을 가르쳐주는거야.
